# Probability and Statistics Foundations

This repository tracks my step-by-step learning journey in Probability and Statistics for Data Science.

I am focusing on intuition first, then formulas, with Python-based practice to connect theory with real data work.

## Learning Approach

- Concepts first, formulas second
- Simple examples before generalization
- Discrete to continuous thinking
- Theory supported by Python simulations
- Clear explanations with comments and examples

## Folder Structure

| Folder | Description |
|--------|-------------|
| `01_probability_basics/` | Sample space, events, and basic probability rules |
| `02_random_variables/` | Discrete and continuous random variables |
| `03_distributions/` | Common probability distributions |
| `04_descriptive_statistics/` | Mean, variance, standard deviation, and shape |
| `05_probability_theory_tools/` | Expectation, covariance, and correlation |
| `06_simulation_and_intuition/` | Simulations to build intuition |
| `07_exercises/` | Practice problems and self-checks |

## Tools Used

- Python
- NumPy
- Matplotlib (when needed)
- Jupyter Notebooks

## Goal

Build strong statistical foundations and apply them confidently to machine learning and data analysis.
